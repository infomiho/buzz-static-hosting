Buzz

Fast static site hosting with GitHub authentication

1

Install the CLI

npm install -g @infomiho/buzz-cli
2

Connect to this server

buzz config server https://{{DOMAIN}}
3

Login with GitHub

buzz login
4

Deploy your site

Deploy any directory containing static files:

buzz deploy ./dist --subdomain my-site

Your site will be live at https://my-site.{{DOMAIN}}
